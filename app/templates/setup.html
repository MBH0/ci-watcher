Webhooks

Configura un webhook en cualquiera de tus repos para que CI Watcher haga build automático en cada push.

Instrucciones:

  1. Selecciona un repo de la lista abajo
  2. Click en "Activar webhook"
  3. GitHub creará un webhook que apunta a {{ host_url }}/api/webhook
  4. En cada push, CI Watcher clonará el repo y hará docker build
  5. Los logs se muestran en tiempo real desde la página de builds
